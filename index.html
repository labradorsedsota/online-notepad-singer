关于

在线记事本 v1.0

一个打开即用的浏览器端记事工具。无需注册，无需后端，笔记保存在浏览器本地。

支持 TXT / Markdown / HTML 三种格式导出。

GitHub →

使用帮助

📝 记录 — 输入标题和内容即可开始记录

💾 保存 — 点击保存按钮，笔记存储在浏览器中，刷新不丢失

📤 导出 — 支持导出为 TXT、Markdown、HTML 文件

🗑 清空 — 清除所有内容，重新开始

🌙 暗色模式 — 右上角开关切换亮/暗主题

🔤 字体 — 右上角下拉菜单切换编辑字体
